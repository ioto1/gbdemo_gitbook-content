Project Delta — FR3 Product Manual

Complete operating and software documentation for the Franka FR3 robot arm.

Overview

Project Delta hosts the official product manual for the Franka FR3 — covering everything from installation and safety instructions to advanced software control interfaces.

The manual is structured into two main sections:

  • Operating Manual — Physical robot setup: safety, mounting, installation, wiring, maintenance, disposal
  • Software Manual — Software ecosystem: Franka UI, FCI real-time API, MATLAB Toolbox, Desk API, RIDE/RACE

Key Features

  • Comprehensive safety documentation — application-related dangers, safety peripherals, rescue systems
  • Step-by-step installation guides — from site preparation to mounting end effectors
  • Software ecosystem overview — Franka UI (Desk), FCI, MATLAB Toolbox, Desk API, RIDE/RACE
  • Advanced control — real-time 1 kHz control via libfranka, ROS/ROS 2 integration
  • Troubleshooting & service — torque calibration, hardware exchange, log download

Manual Structure

Section Description
Operating Manual Physical robot: safety, mounting, wiring, maintenance
Software Manual Software: UI, APIs, control interfaces, troubleshooting

Software Version

This manual covers software version 5.8.0 (hardware version FR3 1.5).
