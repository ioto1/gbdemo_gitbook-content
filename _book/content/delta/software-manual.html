Software Manual

Software Manual — FR3

Complete software ecosystem: Franka UI, FCI real-time API, MATLAB Toolbox, Desk API, and more.

System Overview

The Franka software ecosystem bridges robot hardware, user interfaces, real-time control, and research tools. See the system overview diagram.

Interfaces

Franka UI (Desk)

Browser-based graphical interface for intuitive robot control with drag-and-drop Apps. Ideal for quick setups, demos, and human-robot interaction studies.

Franka Control Interface (FCI)

Low-level real-time API for advanced users — direct control of joint torques, positions, and velocities at 1 kHz. Works with libfranka (C++), ROS/ROS 2, and MATLAB/Simulink.

Franka Toolbox for MATLAB

MATLAB and Simulink tools for algorithm development and testing with real-time robot data.

Desk API

RESTful API for programmatic access — open brakes, manage configurations, download logs.

RIDE & RACE

  • RIDE: Robot Integrated Development Environment for custom Apps
  • RACE: C-library for connecting third-party hardware to the robot
