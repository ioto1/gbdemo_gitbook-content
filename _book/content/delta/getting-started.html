Getting Started

Getting Started — FR3

Prerequisites

  • FR3 robot arm with hardware version compatible with software 5.8.0
  • Ubuntu 22.04+ or Windows 10+ for Desk/FCI
  • Network connection to the robot (Ethernet recommended)

Quick Start

1. Unboxing & Setup

Follow the Operating Manual for physical installation:

  • Prepare installation site
  • Mount the arm
  • Connect power and network
  • Install safety peripherals

2. Access Franka UI

  1. Connect your computer to the robot's network
  2. Open a browser and navigate to the robot's IP address
  3. Log in with default credentials

3. First Movement

From Franka UI (Desk):

  1. Navigate to Apps
  2. Select Test and Jog
  3. Use the jog controls to move each joint
  4. Verify smooth movement without obstructions

4. Advanced Control (FCI)

For programmatic control at 1 kHz:

# Install libfranka
git clone https://github.com/frankaemika/libfranka.git
cd libfranka
mkdir build && cd build
cmake ..
make -j$(nproc)
sudo make install

# Run example
cd examples
./echo_robot_state <robot-ip>

5. MATLAB Integration

% Connect to the robot
robot = franka(<robot-ip>);

% Read joint positions
q = robot.jointPositions;

% Move to home position
robot.moveToHome();
